> [!WARNING]

> * requires root rights

> [!TIP]  
> If curl is not found (command not found), install it from the repository:
> ```shell
> apt install curl -y
> ```
> or you can use wget
> ```shell
> wget <link_to_script> && bash <script_name>
> ```
