## Download script
```shell
curl -o ntlfw-install.sh https://raw.githubusercontent.com/atokarev-tech/ntl-firewall/refs/heads/main/ntl_firewall_install.sh && bash ntlfw-install.sh
```
If curl is not found (command not found), install it from the repository:
```shell
apt install curl
```
⚠️ tested on debian 13 only
